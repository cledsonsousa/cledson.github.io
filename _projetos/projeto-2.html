Projeto Institucional Brasileiro de Iniciação Científica (PIBIC). Assim o CNPQ denomina o programa de concessão de bolsas, tipicamente de 12 meses, a estudantes da graduação para se integrarem de algum modo à pesquisa científica.
Entre os objetivos específicos do programa estão:

  • despertar vocação científica;
  • contribuir para a formação de recursos humanos para a pesquisa;
  • contribuir para a formação científica de recursos humanos que se dedicarão a qualquer atividade profissional;
Este que vos escreve tem especial predileção por este último item. Já que é básico pre-req para qualquer um que queira/precise pôr uma prata no bolso pós formatura. Então, se você quer ser um* engenheir* melhor, pôr uma prata no bolso e chegar no mercado com um pouco mais de experiência, venha participar do nosso Projeto de Iniciação Científica.   • A opulenta bolsa de R$ 525,00. 😒
